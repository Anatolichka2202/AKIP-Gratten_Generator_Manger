# Design

Папка для макетов, дизайн-концептов и ассетов UI.

## Планируемое содержимое

- `mockups/` — Figma-экспорт / скриншоты вайрфреймов
- `assets/` — иконки, изображения, шрифты
- `style/` — QSS-темы для Qt

## Концепция UI

### Главное окно
- Компактное окно (≈800×600)
- Левая панель: подключение + лог
- Правая панель: управление каналами (tabbed или stacked)
- Статус-бар: тип устройства, IP/порт, время последней операции

### Страница АКИП-3417
- Два канала (A и B) рядом
- Для каждого: частота, амплитуда (с единицами), форма сигнала, выход ON/OFF
- AM-модуляция (скрываемая панель)

### Страница Gratten GA1483
- Один RF-канал
- Частота (250 кГц — 4 ГГц)
- Амплитуда (−136 … +13 dBm)
- RF Output ON/OFF
- AM-модуляция

### Диалог настроек
- Вкладка Gratten: IP-адрес, порт (по умолчанию 5025), таймаут
- Вкладка Общие: язык (ru/en), тема
