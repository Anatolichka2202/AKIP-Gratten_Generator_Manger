# Отчёт физика — приборы и их природа

## Gratten GA1483

**Тип**: RF (радиочастотный) генератор сигналов  
**Каналов**: **1 (один)** — нет понятия "канал A / канал B"  
**Диапазон**: 250 kHz — 4 GHz  
**Выход**: ВЧ-сигнал (RF), амплитуда в **dBm** (от −136 до +13 dBm)  
**Интерфейс**: LAN (Ethernet), TCP порт **5025** (стандартный SCPI)  
**Протокол**: SCPI, совместим с RIGOL DSG800 / Atten Technology  

### Команды (краткая выборка)
| Функция | Set | Query |
|---|---|---|
| Несущая частота | `:FREQuency:CW 1e9Hz` | `:FREQuency:CW?` |
| Уровень мощности | `:POWer:LEVEL -10DBM` | `:POWer:LEVEL?` |
| RF выход | `:OUTPut:STATE ON` | `:OUTPut:STATE?` |
| AM модуляция | `:AM:STATE ON` | `:AM:STATE?` |
| FM девиация | `:FM:DEViation 100000` | `:FM:DEViation?` |
| PM девиация | `:PM:DEViation 1.57` | `:PM:DEViation?` |
| Свип старт | `:SWEep:STARt 1e8Hz` | `:SWEep:STARt?` |

### Применение
Тестирование RF-цепей: приёмников, антенн, усилителей ВЧ-диапазона.

## АКИП-3417

**Тип**: DDS генератор функций (AWG — arbitrary waveform generator)  
**Каналов**: **2 (два)** — канал A и канал B, независимые  
**Диапазон**: ~0.1 Hz — 40 MHz (функциональные сигналы)  
**Выход**: SIN, SQU, RAMP, PULSE, DC; амплитуда в **Vpp** (вольт пик-пик)  
**Интерфейс**: USB через драйвер **CH375** (только Windows, проприетарный)  
**Протокол**: SCPI-подобный, совместим с «Suin TFG»

### Команды (краткая выборка)
| Функция | Set | Query |
|---|---|---|
| Частота канала A | `FREQ:CHA 1000` | `FREQ:CHA?` |
| Частота канала B | `FREQ:CHB 1000` | `FREQ:CHB?` |
| Выход канала A | `OUTP:CHA ON` | `OUTP:CHA?` |
| Амплитуда канала A | `VOLT:CHA 1.0 VPP` | `VOLT:CHA?` |
| Форма канала B | `FUNC:CHB SIN` | `FUNC:CHB?` |
| AM состояние | `AM:STAT ON` | *(уточнить с железом)* |

### Применение
Настройка аналоговых цепей: фильтры, усилители, проверка АЧХ в диапазоне до ~40 МГц.

## Ключевые различия для разработки

| Параметр | Gratten GA1483 | АКИП-3417 |
|---|---|---|
| Каналов | **1** | **2 (A + B)** |
| Диапазон | 250 kHz – 4 GHz (RF) | 0.1 Hz – 40 MHz (функциональный) |
| Единица амплитуды | **dBm** | **Vpp** |
| Транспорт | LAN TCP:5025 | USB CH375 (Windows only) |
| Платформа | Кросс-платформенный | Windows only (CH375 SDK) |
| Модуляции | AM, FM, PM, Sweep, PULM, LF | AM, FM (частично); Sweep, Burst (стабы) |
| Команды channel | **не нужны** (`Q_UNUSED`) | обязательны (`:CHA` / `:CHB`) |

### Вывод
Приборы **разного класса**:
- Граттен — высокочастотный RF-генератор (телеком, радиотехника)
- АКИП — низкочастотный функциональный генератор (электроника, измерения)

В коде они объединены через `IAkipController` (общий полиморфный интерфейс), что архитектурно верно, но имя интерфейса вводит в заблуждение — он управляет не только АКИП.
